# ✍️ Spelling Correction (Jaccard Similarity)

## 🔎 What it does
Corrects misspelled words by finding the **closest matching word** from a vocabulary.

---

## 🧠 How it works
- Break words into character sets (or n-grams)  
- Compare similarity between words  
- Pick the best match using **Jaccard Similarity**  

---

## ⚙️ Formula
Similarity = (Common elements) / (Total unique elements)

---

## 🛠️ Stack

- NumPy
